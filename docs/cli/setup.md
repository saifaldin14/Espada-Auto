# `espada setup`

Initialize `~/.espada/espada.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
espada setup
espada setup --workspace ~/espada
```

To run the wizard via setup:

```bash
espada setup --wizard
```
